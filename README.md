# Hotkeys
Lightweight hotkeys javascript

## Get started
```js
<script src="hotkeys/hotkeys.js"></script>
<script type="text/javascript">
hotkeys("ctrl+u", function(){
  // the user clicked on ctrl + u (source code)
  console.log("You've used our shortkey!");
});
</script>
```
